壁纸文件夹使用说明
=================

这个文件夹用于存放启动器的壁纸图片文件。

请将壁纸图片(.jpg, .jpeg, .png格式)放置在此目录下，它们将自动显示在壁纸选择器中。

壁纸命名规则:
- 使用小写字母
- 单词之间用下划线(_)连接
- 例如: nature_mountain.jpg, abstract_blue.png

壁纸文件会根据文件名显示在壁纸选择器中，文件名中的下划线会被替换为空格，
并且首字母会自动大写。

例如:
- nature_mountain.jpg → "Nature mountain"
- abstract_blue.png → "Abstract blue"
